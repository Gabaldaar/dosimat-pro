# **App Name**: ChloriTrack Pro

## Core Features:

- Autenticación Segura y Control de Acceso: El administrador puede iniciar sesión con Google, gestionar otras cuentas de usuario y asignar roles para habilitar funcionalidades específicas, asegurando que solo el personal autorizado pueda realizar acciones sensibles como ajustes financieros o gestión del catálogo de productos.
- Gestión Integral de Clientes: Mantener una base de datos detallada de clientes incluyendo información de contacto con botones de acción rápida (llamar, mapas, email) y datos específicos de piscinas/equipos. Presenta una vista de 'ficha' optimizada para dispositivos móviles.
- Catálogo de Productos y Servicios: Funcionalidad exclusiva del administrador para gestionar una lista dinámica de productos y servicios, permitiendo establecer precios en moneda local (ARS) o USD, con soporte futuro para la carga de imágenes de productos.
- Cuentas Financieras y Seguimiento de Gastos: Gestionar varias cuentas financieras (cuentas bancarias, cajas de efectivo) en múltiples monedas (ARS, USD) y categorizar gastos empresariales, proporcionando acceso rápido a los saldos actuales.
- Entrada Rápida de Transacciones: Una interfaz optimizada para registrar rápidamente todas las transacciones comerciales, incluyendo ventas, reposiciones de cloro, servicios técnicos, pagos de clientes, transferencias entre cuentas y ajustes financieros (solo para administradores).
- Monitoreo Financiero Automatizado y Reportes: Mostrar reportes en tiempo real e indicadores visuales para clientes con deudas pendientes y cuentas financieras con saldos negativos, proporcionando una visión rápida de la salud financiera crítica.
- Herramienta de Notificaciones Inteligentes: Una herramienta potenciada por IA para automatizar el envío de notificaciones personalizadas para eventos como pagos de clientes vencidos o próximas fechas de reposición de cloro, entregadas a usuarios designados según reglas predefinidas.

## Style Guidelines:

- Color principal: Un azul claro y vibrante (#1C7EE0) que simboliza confianza y profesionalidad, ideal para mostrar información clave y elementos interactivos.
- Color de fondo: Un gris azulado muy claro y sutil (#EBEDF4) que proporciona un lienzo limpio, espacioso y altamente legible para el contenido en todos los tamaños de dispositivo.
- Color de acento: Un azul cielo brillante y claro (#8FE4FC) utilizado para llamar la atención sobre llamadas a la acción importantes o elementos destacados, análogo al primario pero distintivamente contrastante.
- Fuente del cuerpo y titulares: 'Inter' (sans-serif), elegida por su excepcional legibilidad en diversas pantallas y su estética moderna y neutra, adecuada para una PWA profesional.
- Utilizar un conjunto consistente de iconos modernos e intuitivos basados en vectores que comuniquen claramente la funcionalidad, mejorando la usabilidad especialmente en interfaces móviles.
- Diseño responsivo 'mobile-first' que presenta diseños basados en tarjetas para listas (clientes, transacciones) y formularios simplificados para la entrada rápida de datos. La navegación debe ser prominente pero discreta.
- Implementar animaciones sutiles y con propósito para transiciones y retroalimentación en las acciones del usuario, asegurando una experiencia fluida y receptiva sin sacrificar el rendimiento, particularmente en dispositivos móviles.
